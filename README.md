# \---

# 

# \## What it does

# 

# Simulates visual experiences sometimes reported by people with dyslexia — directly in the browser. Toggle effects on/off to see how reading feels different for millions of people.

# 

# \*\*8 effects included:\*\*

# \- Letter Jitter — letters shift and wobble

# \- Character Swap — similar letters flip or mirror

# \- Visual Blur — text appears soft and hard to focus

# \- Word Motion — words gently drift

# \- Low Contrast — text fades into the background

# \- Crowding — letters compress together

# \- River Effect — spaces create distracting visual channels

# \- Dyslexia-Friendly mode — shows typography best practices

# 

# \## Bookmarklet

# 

# The site includes a drag-to-bookmark button that applies the simulator to any website — no install needed.

# 

# \## Important context

# 

# Dyslexia is primarily a cognitive difference in language processing — not a vision disorder. This tool simulates a few commonly described visual experiences for empathy and awareness only. Every person's experience is different.
